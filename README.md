
## Simple

> A TypeScript simple template

## 使用

```bash
# 安装依赖
$ npm install

# 启动服务
$ npm start  # visit http://localhost:3333
```

[More docs](https://ice.work/docs/guide/about).

## 注册组件
```js
 <CreateForm schema={schema} components={components} />
 // schema 表示描述组件格式
 // components 表示需要注册的组件
```
